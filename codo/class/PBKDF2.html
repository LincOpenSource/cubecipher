Class: PBKDF2

Defined in: src/pbkdf2.iced

Overview

Standard PBKDF2, as per RFC 2898 and/or PKCS #5 v2

Instance Method Summary

  • # (void) _PRF(input) Private A call to the inner PRF, which is typically HMAC, and fits the HMAC APi.
  • # (void) _gen_T_i({salt, i, progress_hook}, cb) Private Generate one output block of key material, where the output block size is determined by the HMAC's output size.
  • # (void) run({key, salt, dkLen, progress_hook}, cb) Generate dkLen worth of key material.

Constructor Details

# (void) constructor({klass, c})

Parameters:

  • key ( WordArray ) — Will be destroyed after it's used
  • salt ( WordArray )
  • c ( Number ) — the number of iterations
  • dkLen ( Number ) — the needed length of output data
  • klass ( Class ) — The Class of the HMAC to use. Default it is HMAC-SHA512.

Instance Method Details

# (void) _PRF(input) Private

A call to the inner PRF, which is typically HMAC, and fits the HMAC APi.

@oaram WordArray input The input to transform via HMAC

# (void) _gen_T_i({salt, i, progress_hook}, cb) Private

Generate one output block of key material, where the output block size is determined by the HMAC's output size.

Parameters:

  • i ( Number ) — The number of this block (start at 1 and go up..)
  • progress_hook ( Function ) — A standard progress hook for the UI
  • cb ( callback ) — Calls back with one block of key material.

# (void) run({key, salt, dkLen, progress_hook}, cb)

Generate dkLen worth of key material.

Parameters:

  • dkLen ( Number ) — The total length of the derived key material (in bytes)
  • progress_hook ( Function ) — A standard progress hook for the UI
  • cb ( callback ) — Calls back with a WordArray of key-material.
